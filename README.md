# DecodeLabs Internship 

## Student Grade Portal

A full-stack student dashboard application built using HTML, CSS, JavaScript, Node.js, Express, and MongoDB.

---

## Features

- Create new subjects and grades
- View all subjects dynamically
- Edit grades in real time
- Delete subjects in real time
- Responsive navigation menu
- Dynamic card rendering
- RESTful API implementation
- MongoDB database integration

---

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

---

## API Endpoints

### GET
```
GET /api/subjects
```

Returns all subjects.

### POST
```
POST /api/subjects
```

Creates a new subject.

### PUT
```
PUT /api/subjects/:id
```

Updates a grade.

### DELETE
```
DELETE /api/subjects/:id
```

Deletes a subject.
